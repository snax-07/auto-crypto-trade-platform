## Getting Started

### Prerequisites

* **Programming Languages:** Python, Go, JavaScript
* **Package Managers:** Pip, Go modules, NPM
* **Container Runtime:** Docker

### Installation

Clone the repository:

```bash
git clone https://github.com/snax-07/auto-crypto-trade-platform
cd auto-crypto-trade-platform
```

**Using Docker:**

```bash
docker build -t snax-07/auto-crypto-trade-platform .
```

**Using Go Modules:**

```bash
go build
```

**Using NPM:**

```bash
npm install
```

**Using Pip:**

```bash
pip install -r engines/signalEngine/requirements.txt
```

### Usage

**Docker:**

```bash
docker run -it {image_name}
```

**Go Modules:**

```bash
go run {entrypoint}
```

**NPM:**

```bash
npm start
```

**Pip:**

```bash
python {entrypoint}
```

### Testing

**Docker:**

```bash
echo 'INSERT-TEST-COMMAND-HERE'
```

**Go Modules:**

```bash
go test ./...
```

**NPM:**

```bash
npm test
```

**Pip:**

```bash
pytest
```
